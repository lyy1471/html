引言

「宝贝积分」（以下简称「本应用」）重视您的个人信息保护。本《隐私政策》说明我们如何收集、使用、存储、共享与保护您的个人信息，以及您享有的权利。

本应用主要面向家长或监护人，用于家庭场景下的积分与任务管理。请您在监护人陪同下阅读本政策；若您不同意本政策，请勿使用本应用。

一、我们直接收集的信息

本应用以设备本地存储为主，不要求注册账号。我们可能处理的信息包括：

  • 您主动录入的孩子昵称、头像（本地照片）、积分与任务记录；
  • 您开启的功能所产生的本地备份文件（如适用）；
  • 您通过系统权限提供的相机、相册、通知等能力所涉及的数据（均在设备本地或按系统规则处理）。

除为实现您主动选择的备份、分享等功能外，我们不会将上述子女信息上传至我们的自有服务器。

二、设备标识与第三方 SDK

为保障服务稳定、统计使用情况、展示开屏广告并满足合规要求，本应用在您点击「同意」《隐私政策》后，才会初始化下列第三方 SDK。相关 SDK 可能收集设备信息，其中包括 Android ID（安卓设备标识符）。

我们要求第三方在合法、正当、必要范围内处理信息。下列说明收集目的、方式与范围（具体以 SDK 提供方最新公示为准）。

说明：下列 Android ID 相关说明主要适用于安卓版本；iOS 版本在您同意后亦可能初始化友盟统计 SDK，可能收集 IDFA 等设备标识（以 SDK 实际采集为准）。

三、系统权限

为实现功能，本应用可能申请以下权限（以实际系统为准）：

  • 相机 / 相册 / 媒体：设置孩子头像、选择图片；
  • 通知：任务提醒；
  • 存储 / 媒体访问：本地备份与文件导入导出；
  • 网络：连接应用商店、加载隐私政策、SDK 联网统计与广告。

您可在系统「设置」中随时关闭权限；关闭后相关功能可能不可用。

四、信息的存储与保留

  • 您录入的业务数据默认保存在本设备，卸载应用可能被清除；
  • 第三方 SDK 在其服务器上的保留期限以其隐私政策为准；
  • 我们不会在境内服务器长期存储您孩子的积分明细，除非您主动使用需联网的备份能力（如 iOS iCloud，以平台为准）。

五、信息共享与对外提供

我们不会出售您的个人信息。仅在以下情形可能共享：

  • 经您明确同意；
  • 为履行法律法规义务、司法或行政要求；
  • 与上列第三方 SDK 服务提供方共享为实现功能所必要的信息。

安卓版本集成友盟统计与广告 SDK，故可能向友盟提供含 Android ID 在内的设备信息。

六、您的权利

您有权：

  • 查询、更正、删除本设备上的本地数据（通过应用内功能或卸载应用）；
  • 撤回同意：安卓首次启动可选择「暂不同意」并退出；已同意后可在系统设置中清除应用数据并重新选择；
  • 限制追踪：在系统设置中管理广告标识与权限；
  • 就第三方 SDK 处理信息向友盟行使权利，参见其官网隐私政策。

如您对本政策有疑问，请发送邮件至 super_soft@outlook.com，我们将在合理期限内回复。

七、未成年人保护

本应用供监护人管理未成年人日常记录之用。请监护人指导未成年人使用，并妥善保管设备与权限。我们不会主动要求儿童提供超出功能所需的个人信息。

八、政策更新

我们可能适时修订本政策，并通过应用内提示或更新版本说明。重大变更将再次征求您的同意（如法律法规要求）。请您定期查阅本政策。
