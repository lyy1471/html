引言

「宝贝积分」（以下简称「本应用」）重视您的个人信息保护。本《隐私政策》说明我们如何收集、使用、存储、共享与保护您的个人信息，以及您享有的权利。

本应用主要面向家长或监护人，用于家庭场景下的积分与任务管理。请您在监护人陪同下阅读本政策；若您不同意本政策，请勿使用本应用。

一、我们直接收集的信息

本应用以设备本地存储为主，不要求注册账号。我们可能处理的信息包括：

  • 您主动录入的孩子昵称、头像（本地照片）、积分与任务记录；
  • 您开启的功能所产生的本地备份文件（如适用）；
  • 您通过系统权限提供的相册、通知等能力所涉及的数据（均在设备本地或按系统规则处理）。

除为实现您主动选择的备份、分享等功能外，我们不会将上述子女信息上传至我们自有的服务器。在安卓版本，若您已购买高级版并主动使用「云存储备份」，您录入的积分与任务等数据经压缩编码后，将上传至华为 AppGallery Connect 云存储（由华为提供托管，详见下文 2.4）。

二、设备标识与第三方 SDK

为保障服务稳定、统计使用情况、展示开屏广告、提供安卓高级版购买与云备份，并满足合规要求，本应用在您点击「同意」《隐私政策》后，才会初始化下列第三方 SDK 或华为云服务组件。相关 SDK 可能收集设备信息，其中包括 Android ID（安卓设备标识符）。

我们要求第三方在合法、正当、必要范围内处理信息。下列说明收集目的、方式与范围（具体以 SDK 提供方最新公示为准）。

说明：下列 Android ID 相关说明主要适用于安卓版本；华为 IAP / AGC 组件仅适用于安卓版本；iOS 版本在您同意后亦可能初始化友盟统计 SDK，可能收集 IDFA 等设备标识（以 SDK 实际采集为准）。

三、系统权限

为实现功能，本应用可能申请以下权限（以实际系统为准）：

  • 相册 / 媒体：为孩子选择头像图片（仅相册，不使用相机拍摄）；
  • 通知：任务提醒；
  • 存储 / 媒体访问：本地备份与文件导入导出；
  • 网络：连接应用商店与华为云服务、加载隐私政策、SDK 联网统计与广告。

华为 IAP 与 AGC 云存储依赖网络连通，由 HMS Core 等系统组件提供能力，不额外申请与支付或云存储相关的敏感设备权限。

您可在系统「设置」中随时关闭权限；关闭后相关功能可能不可用。

3.1 设备传感器信息（加速度传感器）

经检测，本应用在安卓版本可能访问设备的加速度传感器。本应用自身积分、任务、备份等核心功能不依赖该传感器；相关读取主要由友盟开屏广告 SDK（见 2.2）在展示广告时触发。

信息类型加速度传感器数据（设备在 X/Y/Z 三轴的加速度数值，不含 GPS 等精确位置信息）。
收集目的 识别异常设备与异常流量，进行广告反作弊；统计广告展示与互动效果；支持部分互动广告创意（如摇一摇类广告，以实际投放素材为准）。
收集方式 在您点击「同意」《隐私政策》后，当开屏广告 SDK 初始化、预加载或展示广告时，通过 Android 系统传感器 API 在设备端读取；本应用不将加速度原始数据保存至本地数据库，相关数据可能由 SDK 经加密等方式传输至友盟及其广告联盟合作方（以 SDK 提供方最新公示为准）。
收集范围 仅涉及运动/重力感应相关的加速度数值，不用于识别具体自然人身份；主要发生在冷启动展示开屏广告期间；已购买高级版（免开屏广告）的用户通常不再触发上述 SDK 流程；iOS 版本若集成广告类 SDK 且实际访问传感器，亦适用同类说明（以实际检测与 SDK 行为为准）。
您的选择 首次启动时可选择「暂不同意」并退出；已同意后可通过购买高级版免除开屏广告，或在系统设置中管理应用权限（若系统提供相关选项）。

四、信息的存储与保留

  • 您录入的业务数据默认保存在本设备，卸载应用可能被清除；
  • 第三方 SDK 在其服务器上的保留期限以其隐私政策为准；
  • 我们不会在自有服务器长期存储您孩子的积分明细；
  • iOS：若您主动使用 iCloud 备份，数据按 Apple iCloud 规则存储；
  • Android：若您已购买高级版并主动使用云存储备份，备份文件存储于华为 AppGallery Connect 云存储，卸载应用不会自动删除云端备份，您可在应用内删除或联系我们。

五、信息共享与对外提供

我们不会出售您的个人信息。仅在以下情形可能共享：

  • 经您明确同意；
  • 为履行法律法规义务、司法或行政要求；
  • 与上列第三方 SDK 服务提供方共享为实现功能所必要的信息。

安卓版本集成友盟统计与广告 SDK，故可能向友盟提供含 Android ID 在内的设备信息；在您使用高级版购买、云备份时，还可能向华为提供订单校验、匿名标识及您主动上传的备份数据（以实现对应功能所必要为限）。

六、您的权利

您有权：

  • 查询、更正、删除本设备上的本地数据（通过应用内功能或卸载应用）；
  • 撤回同意：安卓首次启动可选择「暂不同意」并退出；已同意后可在系统设置中清除应用数据并重新选择；
  • 限制追踪：在系统设置中管理广告标识与权限；
  • 就第三方 SDK 处理信息向友盟或华为行使权利，参见其官网隐私政策；
  • 删除云端备份：可在应用内「设置」删除云存储备份，或通过邮件联系我们。
